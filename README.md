# TFFS

TFFS é uma aplicação full stack de gerenciamento de tarefas desenvolvida como projeto de portfólio.

O projeto foi criado para praticar a construção de uma aplicação completa, com frontend, backend, API e banco de dados.

## Funcionalidades

- Criar tarefas
- Listar tarefas
- Editar título de tarefas
- Marcar tarefas como concluídas ou pendentes
- Excluir tarefas
- Exibir contador de tarefas totais, pendentes e concluídas

## Tecnologias utilizadas

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express

### Banco de dados
- SQLite
- Prisma ORM
